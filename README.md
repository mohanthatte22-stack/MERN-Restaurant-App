\# MERN Restaurant App



A full-stack Restaurant and Food Ordering Web Application built using Next.js, Node.js, Express.js, MongoDB, and Tailwind CSS.



\## About the Project



This project is a full-stack restaurant application with frontend and backend integration. It provides food browsing, user authentication, cart management, order management, and admin CRUD functionalities.



\## Technologies Used



\- Next.js

\- React.js

\- Node.js

\- Express.js

\- MongoDB Atlas

\- Tailwind CSS

\- Material UI

\- Redux Toolkit

\- Cloudinary

\- JWT Authentication

\- bcrypt



\## Project Features



\### General Features



\- User registration and login using JWT authentication

\- User information stored in MongoDB

\- File-based routing using Next.js

\- Admin dashboard for food management

\- Dynamic food data fetched from the database

\- Filter food items based on categories

\- Add and remove food items from cart

\- State management using Redux Toolkit

\- Store order details in the database



\### Customer Features



\- Modern and responsive user interface

\- Browse different food categories

\- Register a new account

\- Login to the application

\- Add and remove food from cart

\- Place food orders after authentication



\### Admin Features



\- Protected Admin Dashboard

\- Add new food items

\- Upload food images using Cloudinary

\- Update food details

\- Delete food items

\- View registered users

\- Delete user accounts

\- Passwords secured using bcrypt
